# Internship Hours Calculator

A simple web app to calculate how many work days and weeks are needed to complete an internship, based on total hours and hours per shift.

## Current Files

- `index.html` — main HTML page
- `style.css` — styling for the app
- `script.js` — logic for calculating days and weeks

## Future Enhancements

- Dockerize the app for consistent deployments
- Host on AWS EC2
- Set up CI/CD pipeline with GitHub Actions
- Use Terraform to manage infrastructure
- Add load balancing and scaling
- Optional: Kubernetes deployment
